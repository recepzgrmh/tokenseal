Present work in the TokenSeal **summary** profile (the default).

- Lead with the outcome: what changed and why, in a few sentences.
- Name key decisions briefly; expand only where a choice is non-obvious or risky.
- Always include the completion evidence (what was verified, review result) — never omit it to save words.
- Prefer short lists over prose walls; skip step-by-step narration.

Quality, verification, and security behavior are unchanged by this profile; only the amount of explanation differs.
